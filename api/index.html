⚡ WorkBuddy API

开发者工具服务 - 让 AI 也能调用你的工具

📚 API 端点

GET /api/tools

获取所有可用工具列表

GET /api/tools/:id

获取指定工具的详细信息

GET /api/tools/ai

获取 AI 可调用的工具清单 (OpenAI/OAI 格式)

POST /api/exec/:toolId

执行指定工具

📖 使用示例

curl 调用示例

# 获取所有工具
curl https://your-domain.com/api/tools

# 执行 JSON 格式化
curl -X POST https://your-domain.com/api/exec/json-formatter \
  -H "Content-Type: application/json" \
  -d '{"input": "{\"name\":\"test\"}", "action": "format"}'

# 生成密码
curl -X POST https://your-domain.com/api/exec/password-gen \
  -H "Content-Type: application/json" \
  -d '{"length": 16, "count": 5}'

JavaScript 调用示例

// 在浏览器或 Node.js 中
const response = await fetch('/api/exec/json-formatter', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({
    input: '{"name":"test","value":123}',
    action: 'format',
    indent: 2
  })
});
const result = await response.json();
console.log(result);

🤖 AI 集成

使用 /api/tools/ai 端点获取 OpenAI 格式的工具定义，然后在你的 AI Agent 中使用：
