# 图片工具箱

> 图片压缩、格式转换、Base64 转换

## 功能特性

- **图片压缩** - 减小文件体积
- **格式转换** - PNG/JPG/WebP 互转
- **Base64** - 图片转 Base64 编码
- **尺寸调整** - 调整图片大小
- **本地处理** - 不上传图片，保护隐私

## 快速开始

### 图片压缩

1. 上传或拖拽图片
2. 选择压缩质量
3. 下载压缩后的图片

### 格式转换

1. 上传图片
2. 选择目标格式
3. 下载转换后的图片

## 支持的格式

| 格式 | 特点 |
|------|------|
| PNG | 无损，支持透明 |
| JPG | 有损，体积小 |
| WebP | 现代格式，压缩率高 |

## 常见问题

**Q: 压缩会影响图片质量吗？**
A: 有一定影响，可调节压缩比例。

**Q: 什么情况下用 WebP？**
A: WebP 在同等质量下文件更小，适合网页使用。

**Q: 图片会发送到服务器吗？**
A: 不会，所有处理在浏览器本地完成。
