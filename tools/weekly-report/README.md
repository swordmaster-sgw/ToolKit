# 周报智能助手

> 生成结构化的工作周报

## 功能特性

- **结构化录入** - 本周工作、下周计划
- **多种模板** - 不同风格周报模板
- **一键生成** - 快速生成完整周报

## 快速开始

1. 选择周报模板
2. 填写本周完成工作
3. 填写下周计划
4. 填写心得/问题
5. 生成并导出周报

## 周报模板结构

```
# 周报 (2026/03/09 - 2026/03/15)

## 本周完成
- [已完成任务1]
- [已完成任务2]

## 下周计划
- [计划任务1]
- [计划任务2]

## 心得与问题
[可选填写]
```

## 常见问题

**Q: 周报可以导出什么格式？**
A: 支持 Markdown 格式导出。

**Q: 有哪些模板可选？**
A: 提供简明版、详细版、项目版等多种模板。

**Q: 周报数据会保存吗？**
A: 当前版本数据存储在浏览器本地。
