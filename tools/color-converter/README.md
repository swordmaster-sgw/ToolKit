# 颜色转换器

> HEX、RGB、HSL 颜色格式互转

## 功能特性

- **格式互转** - HEX、RGB、HSL 一键转换
- **取色器** - 选择任意颜色
- **渐变生成** - 自动生成渐变代码
- **色板收藏** - 保存常用颜色

## 快速开始

1. 输入任意格式的颜色值
2. 自动转换为其他格式
3. 点击复制按钮获取结果

## 支持的格式

### HEX (十六进制)

- 3位简写：`#FFF`
- 6位完整：`#FFFFFF`
- 带透明度：`#FFFFFFFF`

### RGB

- 标准：`rgb(255, 255, 255)`
- 简化：`rgb(255 255 255)`
- 带透明度：`rgba(255, 255, 255, 1)`

### HSL

- 标准：`hsl(0, 0%, 100%)`
- 带透明度：`hsla(0, 0%, 100%, 1)`

## 常见问题

**Q: HEX 和 RGB 怎么转换？**
A: HEX 是 RGB 的十六进制表示，如 `#FF0000` = `rgb(255, 0, 0)`。

**Q: HSL 颜色怎么理解？**
A: HSL = 色相(0-360°) + 饱和度(0-100%) + 亮度(0-100%)。

**Q: 什么是 RGBA 和 HSLA？**
A: 带透明度版本，最后一个参数是透明度（0-1）。
