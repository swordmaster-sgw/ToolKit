# 每日站会助手

> 记录和生成每日站会汇报

## 功能特性

- **结构化记录** - 昨日/今日/阻塞
- **一键生成** - 生成标准格式汇报
- **历史记录** - 保存历史站会
- **快速复制** - 一键复制汇报内容

## 快速开始

1. 填写「昨日完成」
2. 填写「今日计划」
3. 填写「阻塞问题」（如有）
4. 点击生成站会汇报
5. 复制并在团队会议中分享

## 站会格式

```
## 2026-03-18 站会

### 昨日完成
- 任务 A
- 任务 B

### 今日计划
- 任务 C
- 任务 D

### 阻塞问题
- 无
```

## 常见问题

**Q: 站会有什么用？**
A: 同步团队进度，及早发现和解决阻塞问题。

**Q: 历史记录可以删除吗？**
A: 可以，管理历史记录功能中删除。

**Q: 支持导出什么格式？**
A: 支持 Markdown 格式导出。
