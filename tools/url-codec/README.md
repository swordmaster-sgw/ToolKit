# URL 编解码

> URL 编码解码与参数解析工具

## 功能特性

- **编码/解码** - 完整 URL 或单个参数编码
- **参数解析** - 解析 URL 中的查询参数
- **构建 URL** - 组合 URL 和参数
- **批量操作** - 支持批量处理

## 快速开始

### 编码 URL

1. 输入要编码的 URL 或文本
2. 点击「编码」按钮
3. 特殊字符将被转换为 `%XX` 格式

### 解码 URL

1. 输入编码后的 URL
2. 点击「解码」按钮
3. 获取原始内容

### 解析参数

1. 输入包含查询参数的 URL
2. 点击「解析参数」
3. 查看参数列表和值

## URL 编码说明

URL 中特殊字符需要编码：

| 字符 | 编码 |
|------|------|
| 空格 | `%20` |
| 中文 | UTF-8 编码 |
| `&` | `%26` |
| `=` | `%3D` |
| `?` | `%3F` |

## 常见问题

**Q: 编码和解码有什么区别？**
A: 编码将特殊字符转为安全格式，解码还原原始字符。

**Q: 什么时候需要 URL 编码？**
A: 当 URL 包含中文、空格、特殊字符时需要编码。

**Q: 如何解析 URL 参数？**
A: 工具自动解析 `?key=value&key2=value2` 格式的参数。
