# Markdown 预览

> 实时预览 Markdown 渲染效果

## 功能特性

- **实时预览** - 边写边看效果
- **代码高亮** - 代码块语法着色
- **表格支持** - Markdown 表格渲染
- **数学公式** - LaTeX 数学公式支持
- **导出 HTML** - 下载为 HTML 文件

## 快速开始

1. 在左侧编辑区输入 Markdown
2. 右侧实时预览渲染效果
3. 点击导出按钮下载 HTML

## 支持的语法

### 基础语法

```markdown
# 标题1
## 标题2

**粗体** *斜体*

- 列表项
1. 有序列表

[链接](https://example.com)
![图片](url)
```

### 代码块

<pre>
```javascript
console.log('Hello');
```
</pre>

### 表格

<pre>
| 列1 | 列2 |
|------|------|
| 内容 | 内容 |
</pre>

## 常见问题

**Q: 如何插入代码？**
A: 使用三个反引号包裹代码，并指定语言。

**Q: 支持数学公式吗？**
A: 支持 LaTeX 格式的数学公式。

**Q: 导出的 HTML 可以分享吗？**
A: 可以，导出的 HTML 是独立的文件，可在浏览器中直接打开。
