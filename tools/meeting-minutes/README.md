# 会议纪要生成器

> 快速生成标准格式的会议纪要

## 功能特性

- **结构化录入** - 议题、决策、行动项
- **自动格式化** - 生成标准会议纪要
- **导出** - 支持多种格式导出

## 快速开始

1. 填写会议基本信息
2. 添加会议议题
3. 记录决策
4. 添加行动项（负责人 + 时间）
5. 生成并导出会议纪要

## 会议纪要结构

```
# 会议纪要

## 基本信息
- 时间：
- 参会人：
- 主持人：

## 议题

## 决策

## 行动项
| 事项 | 负责人 | 时间 |
|------|--------|------|
```

## 常见问题

**Q: 行动项是什么？**
A: 会议中分配的具体任务，包括任务内容、负责人、完成时间。

**Q: 可以导出什么格式？**
A: 支持 Markdown 格式导出。

**Q: 如何跟踪行动项？**
A: 可将导出的会议纪要保存，定期跟进完成情况。
