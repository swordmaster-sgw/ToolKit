# HTTP 状态码速查

> 完整 HTTP 状态码参考手册

## 功能特性

- **完整列表** - 1xx-5xx 所有状态码
- **分类清晰** - 按类别分组
- **搜索** - 快速查找
- **处理建议** - 给出处理思路

## 状态码分类

| 类别 | 含义 |
|------|------|
| 1xx | 信息响应 |
| 2xx | 成功响应 |
| 3xx | 重定向 |
| 4xx | 客户端错误 |
| 5xx | 服务器错误 |

## 常用状态码

| 状态码 | 含义 |
|--------|------|
| 200 | OK - 请求成功 |
| 301 | 永久重定向 |
| 302 | 临时重定向 |
| 400 | 请求语法错误 |
| 401 | 未授权 |
| 403 | 禁止访问 |
| 404 | 资源不存在 |
| 500 | 服务器内部错误 |
| 502 | 网关错误 |
| 503 | 服务不可用 |

## 常见问题

**Q: 4xx 和 5xx 有什么区别？**
A: 4xx 是客户端问题（如请求错误），5xx 是服务器问题（如服务崩溃）。

**Q: 301 和 302 有什么区别？**
A: 301 表示永久移动，搜索引擎会更新索引；302 表示临时移动。

**Q: 如何排查 5xx 错误？**
A: 查看服务器日志，定位具体错误原因。
