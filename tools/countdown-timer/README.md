# 项目倒计时

> 项目里程碑和截止日期倒计时

## 功能特性

- **多倒计时** - 同时管理多个项目节点
- **桌面提醒** - 浏览器通知提醒
- **本地存储** - 保存倒计时数据

## 快速开始

1. 添加倒计时事件
2. 设置目标日期和描述
3. 查看倒计时
4. 接收提醒通知

## 时间格式

| 格式 | 说明 |
|------|------|
| XX 天 | 剩余天数 |
| XX:XX:XX | 时:分:秒 |

## 常见问题

**Q: 倒计时会丢失吗？**
A: 数据保存在浏览器本地，清除缓存会丢失。

**Q: 如何添加多个倒计时？**
A: 点击添加按钮，可创建多个倒计时事件。

**Q: 过期后会显示什么？**
A: 会显示「已过期」状态。
