# 命令速查手册

> Git、Docker、Linux 等常用命令速查

## 功能特性

- **多类别** - Git、Docker、Linux 等
- **搜索** - 快速搜索命令
- **收藏** - 收藏常用命令

## 支持的命令

### Git

```bash
git init          # 初始化仓库
git add .         # 添加到暂存区
git commit -m ""  # 提交
git push          # 推送到远程
git pull          # 拉取远程
git branch        # 查看分支
git checkout      # 切换分支
```

### Docker

```bash
docker build      # 构建镜像
docker run        # 运行容器
docker ps         # 查看容器
docker stop       # 停止容器
docker rm         # 删除容器
docker logs       # 查看日志
```

### Linux

```bash
ls -la           # 列出文件
cd               # 切换目录
mkdir            # 创建目录
rm -rf           # 删除文件
grep             # 搜索内容
chmod            # 修改权限
```

## 常见问题

**Q: 命令行太长了记不住怎么办？**
A: 收藏常用命令，下次快速查找。

**Q: 有快捷键吗？**
A: 使用 Ctrl+K 快速搜索命令。
