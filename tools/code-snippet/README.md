# 代码片段管理器

> 收藏和管理常用代码片段

## 功能特性

- **收藏** - 保存常用代码
- **分类** - 按语言/用途分类
- **搜索** - 快速查找片段
- **一键复制** - 复制完整代码

## 快速开始

1. 添加代码片段
2. 选择语言类型
3. 添加标签方便搜索
4. 保存并使用

## 片段管理

- 创建新片段
- 编辑已有片段
- 删除不需要的片段
- 搜索目标片段

## 常见问题

**Q: 代码片段保存在哪？**
A: 存储在浏览器本地存储中。

**Q: 支持哪些语言？**
A: 支持主流编程语言语法高亮。

**Q: 可以导出片段吗？**
A: 支持导出为 JSON 格式备份。
