# Bug 报告生成器

> 生成规范格式的 Bug 报告

## 功能特性

- **结构化** - 完整记录 Bug 信息
- **一键生成** - 生成标准格式报告
- **多平台** - 适配 Jira、GitLab 等平台

## 快速开始

1. 填写 Bug 标题
2. 描述问题现象
3. 列出复现步骤
4. 填写预期结果和实际结果
5. 生成报告并复制

## Bug 报告结构

```
## Bug 标题

### 问题描述
[描述问题]

### 复现步骤
1. 步骤一
2. 步骤二
3. 步骤三

### 预期结果
[应该发生什么]

### 实际结果
[实际发生了什么]

### 环境信息
- 浏览器：
- 操作系统：
- 版本：
```

## 常见问题

**Q: 报告可以提交到哪里？**
A: 可复制到 Jira、GitLab、Redmine 等问题跟踪系统。

**Q: 需要填写哪些信息？**
A: 至少填写标题、描述、复现步骤，其他信息越详细越好。

**Q: 如何提高 Bug 报告质量？**
A: 提供明确的复现步骤、截图、环境信息等。
