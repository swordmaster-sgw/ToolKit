# 时间戳转换

> Unix 时间戳与日期时间的互转工具

## 功能特性

- **实时时钟** - 显示当前时间戳和日期
- **双向转换** - 时间戳 ↔ 日期时间
- **多时区** - 同时显示多个时区时间
- **一键复制** - 点击结果即可复制

## 快速开始

### 时间戳 → 日期

1. 输入 10 位（秒）或 13 位（毫秒）时间戳
2. 系统自动转换为多种格式：
   - 本地时间
   - ISO 8601
   - UTC 时间
   - 毫秒/秒时间戳

### 日期 → 时间戳

1. 选择或输入日期时间
2. 自动获取对应的秒/毫秒时间戳

## 多时区对照

工具内置以下时区实时对照：

- 北京 (UTC+8)
- 东京 (UTC+9)
- 伦敦 (UTC+0)
- 纽约 (UTC-5)
- 洛杉矶 (UTC-8)
- 悉尼 (UTC+11)

## 快捷操作

- 点击「填入当前时间」快速填充当前时间戳
- 点击结果值可直接复制
- 可暂停/恢复时钟更新

## 常见问题

**Q: 如何区分秒和毫秒时间戳？**
A: 10 位数字通常是秒级时间戳，13 位是毫秒级。工具会自动识别。

**Q: 北京时间和 UTC 有什么区别？**
A: UTC 是世界统一时间，北京时间 = UTC + 8 小时。
