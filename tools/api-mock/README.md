# API 接口文档模板

> 快速生成标准 API 接口文档

## 功能特性

- **结构化** - 完整 API 文档结构
- **示例** - 请求/响应示例
- **导出** - 导出 Markdown 格式

## 快速开始

1. 填写接口基本信息
2. 填写请求参数
3. 填写响应格式
4. 添加错误码
5. 生成完整文档

## 文档结构

```
# API 接口文档

## 接口名称
### 基本信息
- 接口地址：
- 请求方式：
-  Content-Type：

### 响应示例
成功：
失败：
```

## 常见问题

**Q: 支持哪些请求方式？**
A: 支持 GET、POST、PUT、DELETE 等常用方式。

**Q: 可以添加多个接口吗？**
A: 可以添加多个接口文档。

**Q: 导出的格式是什么？**
A: 导出为 Markdown 格式。
