# PPT 大纲生成器

> 生成结构化的 PPT 演示大纲

## 功能特性

- **主题输入** - 输入汇报主题
- **自动生成** - 生成 PPT 结构
- **内容建议** - 每页核心要点
- **导出** - 导出为 Markdown

## 快速开始

1. 输入 PPT 主题
2. 填写目标受众
3. 设置页数
4. 生成大纲
5. 导出使用

## 大纲结构

```
# [PPT标题]

## 第1页：开场
- 核心要点1
- 核心要点2

## 第2页：[章节标题]
- 内容要点
- 数据支持
```

## 常见问题

**Q: 生成的大纲可以修改吗？**
A: 可以，生成后可自行调整内容。

**Q: 适合什么场景？**
A: 工作汇报、演讲、培训等需要 PPT 的场景。

**Q: 导出的格式是什么？**
A: 导出为 Markdown 格式，可粘贴到 PPT 制作工具。
