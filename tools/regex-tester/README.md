# 正则表达式测试器

> 实时高亮匹配结果的正则表达式验证工具

## 功能特性

- **实时匹配** - 输入正则即可看到匹配结果
- **语法高亮** - 匹配内容以黄色高亮显示
- **捕获组** - 显示所有捕获组及位置信息
- **预设模板** - 内置常用正则表达式模板

## 快速开始

### 基本使用

1. 在「正则表达式」输入框中输入正则模式
2. 在「测试文本」区域输入待测试内容
3. 匹配结果将实时显示在下方的「匹配结果」区域

### Flags 说明

| Flag | 说明 |
|------|------|
| `g` | 全局匹配（查找所有匹配项）|
| `i` | 忽略大小写 |
| `m` | 多行模式 |

## 预设模板

点击预设按钮可快速填入常用正则表达式：

- 手机号（1[3-9]\\d{9}）
- 邮箱地址
- 身份证号
- IP 地址
- URL
- 日期（YYYY-MM-DD）
- 中文字符
- HTML 标签
- 版本号
- 空白行

## 常见问题

**Q: 正则表达式有语法错误怎么办？**
A: 工具会显示具体错误信息，帮助定位问题。

**Q: 如何匹配中文？**
A: 使用 `[\u4e00-\u9fa5]` 可匹配中文字符。

**Q: 匹配结果太多怎么办？**
A: 工具默认仅显示前 50 条匹配结果。
