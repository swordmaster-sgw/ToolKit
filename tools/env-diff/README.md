# 环境配置对比

> 对比不同环境的配置文件差异

## 功能特性

- **双栏对比** - 左侧开发/右侧生产
- **差异高亮** - 突出显示不同之处
- **配置检测** - 识别缺失的配置项

## 快速开始

1. 粘贴开发环境配置
2. 粘贴生产环境配置
3. 自动对比差异
4. 查看差异报告

## 常见用途

- 开发环境 vs 生产环境
- 测试环境 vs 预发布环境
- 配置文件备份对比

## 常见问题

**Q: 对比哪些配置？**
A: 支持 .env、JSON、YAML 等格式。

**Q: 差异会高亮吗？**
A: 是的，不同之处会红色高亮显示。

**Q: 如何识别缺失项？**
A: 工具会自动检测哪边有额外配置项。
