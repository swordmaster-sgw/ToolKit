# CSV ↔ JSON 转换

> CSV 和 JSON 格式数据的双向转换工具

## 功能特性

- **双向转换** - CSV 转 JSON、JSON 转 CSV
- **自定义分隔符** - 支持多种分隔符
- **表头映射** - 自定义字段名称
- **数据预览** - 转换前预览数据

## 快速开始

### CSV 转 JSON

1. 粘贴 CSV 数据或上传 CSV 文件
2. 选择分隔符（逗号、制表符、分号等）
3. 点击「转换为 JSON」
4. 获取转换结果

### JSON 转 CSV

1. 粘贴 JSON 数据
2. 点击「转换为 CSV」
3. 获取表格数据

## CSV 格式说明

CSV（逗号分隔值）示例：
```csv
姓名,年龄,城市
张三,25,北京
李四,30,上海
```

## 常见问题

**Q: CSV 文件用什么分隔符？**
A: 常用逗号 `,`，也支持制表符 `Tab`、分号 `;` 等。

**Q: JSON 嵌套对象如何处理？**
A: 工具会自动展平嵌套对象。

**Q: 如何处理带逗号的字段？**
A: 字段用引号包裹即可，如 `"Hello, World"`。
