# 文本批处理工具

> 批量处理文本的多种操作

## 功能特性

- **去重** - 删除重复行
- **排序** - 按字母/数字排序
- **计数** - 统计行数、字符数
- **替换** - 批量文本替换
- **大小写** - 大小写转换

## 快速开始

1. 输入或粘贴文本
2. 选择处理操作
3. 点击执行
4. 获取处理结果

## 支持的操作

| 操作 | 说明 |
|------|------|
| 去重 | 删除重复行 |
| 排序 | 按字母/数字排序 |
| 反序 | 行顺序反转 |
| 行数 | 统计行数 |
| 字符数 | 统计字符数 |
| 替换 | 批量替换文本 |
| 大小写 | 转换为大写/小写 |

## 常见问题

**Q: 去重是按什么标准？**
A: 按整行内容去重，完全相同的行为重复。

**Q: 排序支持数字吗？**
A: 支持，可按数字或字母顺序排序。

**Q: 可以组合多个操作吗？**
A: 可以逐个执行多个操作。
