# 番茄钟 & 专注计时

> 基于番茄工作法的时间管理工具

## 功能特性

- **计时器** - 25/5 分钟标准番茄钟
- **任务绑定** - 关联当前任务
- **统计** - 记录每日完成数量
- **音效提醒** - 计时结束提醒

## 番茄工作法

1. 选择一个任务
2. 设置 25 分钟计时
3. 专注工作直到铃响
4. 休息 5 分钟
5. 每 4 个番茄钟长休息 15-30 分钟

## 计时器操作

- 开始/暂停计时
- 重置当前番茄钟
- 跳过休息时间

## 常见问题

**Q: 番茄钟时间可以调整吗？**
A: 可以自定义工作时间和休息时间。

**Q: 休息时需要做什么？**
A: 离开工位，放松身心，可以喝水、走动等。

**Q: 统计数据在哪看？**
A: 页面底部显示今日完成番茄数。
