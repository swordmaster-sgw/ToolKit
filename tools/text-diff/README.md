# 文本 Diff 对比

> 逐行对比两段文本的差异

## 功能特性

- **高亮差异** - 清晰显示新增、删除、修改
- **逐行对比** - 精准定位每行差异
- **代码友好** - 适合代码和配置文件对比

## 快速开始

1. 在左侧输入原始文本
2. 在右侧输入对比文本
3. 差异部分将高亮显示

## 颜色说明

| 颜色 | 含义 |
|------|------|
| 绿色 | 新增的内容 |
| 红色 | 删除的内容 |
| 黄色 | 修改的行 |

## 常见问题

**Q: 如何对比两个文件的差异？**
A: 分别将两个文件内容复制到左右两侧即可对比。

**Q: 适合对比哪些内容？**
A: 代码、配置文件、文档等文本内容。

**Q: 对比结果可以复制吗？**
A: 可以选择复制左侧或右侧的文本内容。
