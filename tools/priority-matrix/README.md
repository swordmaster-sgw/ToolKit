# 需求优先级矩阵

> 基于四象限的需求优先级管理工具

## 功能特性

- **四象限** - 紧急/重要分类
- **拖拽排序** - 直观调整优先级
- **导出** - 导出为任务清单

## 快速开始

1. 添加需求到对应象限
2. 拖拽调整优先级
3. 导出任务清单

## 四象限说明

| 象限 | 说明 | 行动 |
|------|------|------|
| 重要且紧急 | 立即处理 | 马上做 |
| 重要不紧急 | 计划安排 | 计划做 |
| 紧急不重要 | 可委托他人 | 授权做 |
| 不重要不紧急 | 可删除 | 尽量不做 |

## 常见问题

**Q: 如何判断重要和紧急？**
A: 重要 - 影响项目目标；紧急 - 需要立即处理。

**Q: 可以导出什么格式？**
A: 可导出为任务清单格式。

**Q: 数据可以保存吗？**
A: 当前版本数据存储在浏览器本地。
