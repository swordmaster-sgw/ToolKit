# SQL 格式化

> SQL 语句的美化和压缩工具

## 功能特性

- **语句美化** - 自动格式化 SQL
- **压缩** - 去除多余空白
- **关键字高亮** - 彩色显示 SQL 关键字
- **多方言支持** - MySQL、PostgreSQL、标准SQL

## 快速开始

1. 粘贴 SQL 语句到输入区
2. 点击「美化」或「压缩」按钮
3. 获取格式化后的 SQL

## 支持的方言

- MySQL
- PostgreSQL
- 标准 SQL (ANSI)
- SQLite

## 常见 SQL 关键字

```sql
SELECT, FROM, WHERE, JOIN, LEFT, RIGHT
INSERT, UPDATE, DELETE, CREATE, DROP, ALTER
GROUP BY, ORDER BY, HAVING, LIMIT, OFFSET
AND, OR, NOT, IN, LIKE, BETWEEN, IS NULL
```

## 常见问题

**Q: 为什么格式化后和原来不一样？**
A: 格式化会按标准规则重新排版，包括缩进、大小写等。

**Q: 支持哪些 SQL 操作？**
A: 支持 SELECT、INSERT、UPDATE、DELETE、CREATE 等常见 SQL。

**Q: 压缩后会丢失内容吗？**
A: 不会，只是去除多余的空格和换行，保留所有内容。
