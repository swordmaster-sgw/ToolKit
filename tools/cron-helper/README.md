# Cron 表达式助手

> 可视化生成和解析 Cron 表达式

## 功能特性

- **可视化生成** - 图形化界面选择时间
- **实时预览** - 显示下次执行时间
- **双向解析** - 解析现有 Cron 表达式
- **多种格式** - 支持 5 位和 6 位格式

## Cron 格式说明

### 5 位格式（标准）

```
* * * * *
│ │ │ │ │
│ │ │ │ └─ 星期 (0-7, 0和7都是周日)
│ │ │ └─── 月份 (1-12)
│ │ └───── 日期 (1-31)
│ └─────── 小时 (0-23)
└───────── 分钟 (0-59)
```

### 6 位格式（支持秒）

```
* * * * * *
│ │ │ │ │ │
│ │ │ │ │ └─ 秒 (0-59)
│ │ │ │ └─── 星期
│ │ │ └───── 月份
│ │ └─────── 日期
│ └───────── 小时
└─────────── 分钟
```

## 快速开始

1. 选择或输入 Cron 表达式
2. 下方显示下次执行时间
3. 支持常见模板快速选择

## 常用示例

| 表达式 | 含义 |
|--------|------|
| `* * * * *` | 每分钟 |
| `0 * * * *` | 每小时整点 |
| `0 0 * * *` | 每天午夜 |
| `0 0 * * 0` | 每周日 |
| `0 0 1 * *` | 每月1日 |
| `*/5 * * * *` | 每5分钟 |

## 常见问题

**Q: Cron 表达式中的数字怎么填？**
A: 使用具体数字（如 `5`）、范围（如 `1-5`）、列表（如 `1,3,5`）或通配符（如 `*`）。

**Q: 如何理解「下次执行时间」？**
A: 根据当前时间计算，显示接下来 5 次的具体执行时间点。

**Q: 5位和6位有什么区别？**
A: 6位格式支持秒级精度，5位是标准格式不含秒。
