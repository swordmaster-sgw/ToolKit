# 密码生成器

> 生成安全的随机密码

## 功能特性

- **强随机** - 加密安全的随机数
- **可配置** - 长度、字符集
- **批量生成** - 一次生成多个
- **复制** - 一键复制

## 快速开始

1. 设置密码长度
2. 选择字符类型
3. 点击生成
4. 复制使用

## 可选字符

| 类型 | 包含 |
|------|------|
| 大写字母 | A-Z |
| 小写字母 | a-z |
| 数字 | 0-9 |
| 符号 | !@#$%^&* 等 |

## 常见问题

**Q: 生成的密码安全吗？**
A: 使用加密安全的随机数生成，安全性高。

**Q: 长度多少合适？**
A: 推荐 12-16 位，越长越安全。

**Q: 密码会保存吗？**
A: 不会保存，生成后请立即使用并妥善保管。
