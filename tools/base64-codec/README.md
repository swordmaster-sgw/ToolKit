# Base64 编解码

> 文本和文件的 Base64 编码解码工具

## 功能特性

- **文本编码** - 支持中文、Unicode 字符
- **文件编码** - 图片、文档等文件转 Base64
- **URL-safe 模式** - 生成 URL 安全的编码
- **一键复制** - 快速复制结果到剪贴板

## 快速开始

### 文本模式

1. 在「输入文本」区域输入内容
2. 选择编码/解码模式
3. 点击对应按钮执行操作
4. 点击「复制」获取结果

### 文件模式

1. 点击「文件模式」标签
2. 点击或拖拽文件到上传区域
3. 自动生成文件的 Base64 编码
4. 点击复制按钮获取结果

## 选项说明

| 选项 | 说明 |
|------|------|
| URL-safe 模式 | 将 `+/` 替换为 `-_`，适合 URL 传输 |
| 支持中文/Unicode | 勾选后支持非 ASCII 字符编码 |

## 常见问题

**Q: Base64 编码用在哪些场景？**
A: 常用于数据传输、URL 参数、邮件附件、JSON 数据中的二进制内容等。

**Q: 解码失败怎么办？**
A: 请确认输入是有效的 Base64 字符串，可尝试取消勾选「URL-safe 模式」。

**Q: 文件编码有什么用？**
A: 可将图片等文件转为文本格式，便于在 JSON、HTML 中直接使用。
