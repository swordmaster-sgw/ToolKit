# JSON 格式化 / 压缩

> JSON 数据的美化、压缩、校验一体化工具

## 功能特性

- **格式化美化** - 自动缩进、可视化语法高亮
- **压缩** - 一键压缩为单行，节省空间
- **校验** - 实时语法检查，定位错误位置
- **排序** - 按字母顺序排列 JSON Key
- **统计** - 显示键值数量、嵌套深度

## 快速开始

### 格式化 JSON

1. 在左侧输入区域粘贴 JSON
2. 点击「格式化」按钮或按 `Ctrl+Enter`
3. 格式化结果将显示在右侧输出区

### 压缩 JSON

1. 粘贴 JSON 到输入区
2. 点击「压缩」按钮
3. 获取压缩后的单行 JSON

### 校验 JSON

1. 输入或粘贴 JSON 内容
2. 点击「校验」按钮
3. 查看语法是否合法及统计信息

## 快捷键

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+Enter` | 格式化 |
| `Ctrl+Shift+M` | 压缩 |

## 拖拽文件

支持直接将 `.json` 文件拖拽到输入区域，系统将自动读取文件内容。

## 常见问题

**Q: 输入的 JSON 有错误怎么办？**
A: 工具会显示错误位置和具体错误信息，方便定位问题。

**Q: 如何处理大型 JSON 文件？**
A: 建议文件大小不超过 5MB，超出可能影响性能。

**Q: 支持哪种缩进风格？**
A: 可选择 2 空格、4 空格或 Tab 缩进。
