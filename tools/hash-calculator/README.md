# Hash 计算器

> 计算文本和文件的 Hash 摘要值

## 功能特性

- **多种算法** - 支持 MD5、SHA-1、SHA-256、SHA-512
- **文件计算** - 支持拖拽文件计算 Hash
- **本地计算** - 所有计算在浏览器本地完成，不上传数据
- **批量计算** - 文本和文件分别计算

## 快速开始

### 文本 Hash

1. 在「文本输入」区域输入内容
2. 选择 Hash 算法
3. 点击「计算」按钮
4. 查看各算法的结果

### 文件 Hash

1. 切换到「文件」标签
2. 点击或拖拽文件到上传区域
3. 自动计算文件的 Hash 值

## 支持的算法

| 算法 | 输出长度 | 安全性 |
|------|----------|--------|
| MD5 | 32 字符 | 低（已破解）|
| SHA-1 | 40 字符 | 中（已发现碰撞）|
| SHA-256 | 64 字符 | 高 |
| SHA-512 | 128 字符 | 最高 |

## 常见问题

**Q: Hash 值可以解密吗？**
A: Hash 是单向函数，无法解密。只能对比原始数据和 Hash 值是否匹配。

**Q: 为什么文件 Hash 比较慢？**
A: 大文件需要读取全部内容计算，建议 100MB 以内。

**Q: 哪个算法最安全？**
A: 推荐使用 SHA-256 或 SHA-512，MD5 和 SHA-1 已不安全。
