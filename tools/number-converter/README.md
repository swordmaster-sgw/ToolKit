# 进制转换器

> 二进制、八进制、十进制、十六进制互转

## 功能特性

- **格式互转** - 常见进制一键转换
- **负数支持** - 支持负数转换
- **IP 转换** - IP 地址二进制转换
- **实时计算** - 输入即显示结果

## 快速开始

1. 输入任意进制的数字
2. 自动转换为其他进制格式
3. 点击复制获取结果

## 支持的进制

| 进制 | 基数 | 示例 |
|------|------|------|
| 二进制 | 2 | 1010 |
| 八进制 | 8 | 12 |
| 十进制 | 10 | 10 |
| 十六进制 | 16 | A |

## 常见问题

**Q: 不同进制有什么区别？**
A: 计算机中不同进制表示同一数值，如十进制 255 = 十六进制 FF = 二进制 11111111。

**Q: 十六进制中 A-F 是什么？**
A: A=10, B=11, C=12, D=13, E=14, F=15。

**Q: 如何快速转换 IP 地址？**
A: 输入 IP 地址，自动显示二进制表示。
