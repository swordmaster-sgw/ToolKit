# 数据脱敏工具

> 对敏感数据进行批量脱敏处理

## 功能特性

- **多种类型** - 手机号、身份证、邮箱、银行卡
- **批量处理** - 一次处理多条数据
- **自定义规则** - 可调整脱敏规则
- **本地处理** - 数据不上传，保护隐私

## 快速开始

1. 选择要脱敏的数据类型
2. 输入或粘贴待脱敏的数据
3. 点击「脱敏处理」
4. 获取脱敏后的结果

## 支持的类型

| 类型 | 示例 | 脱敏后 |
|------|------|--------|
| 手机号 | 13800138000 | 138****8000 |

| 银行卡 | 6222021234567890 | 6222 **** **** 7890 |

## 常见问题

**Q: 脱敏后数据还能用吗？**
A: 脱敏是不可逆的，仅用于展示或测试，不能还原为原始数据。

**Q: 脱敏规则可以修改吗？**
A: 部分类型支持自定义保留字符数。

**Q: 数据会发送到服务器吗？**
A: 不会，所有处理在浏览器本地完成。
