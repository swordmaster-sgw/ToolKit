# 日志分析工具

> 分析和过滤日志内容

## 功能特性

- **关键字高亮** - 突出显示关键信息
- **级别分类** - 错误/警告/信息分类
- **过滤** - 按关键字过滤日志
- **统计** - 统计各类日志数量

## 快速开始

1. 粘贴日志内容
2. 设置过滤关键字
3. 选择日志级别
4. 查看分析结果

## 日志级别

| 级别 | 标识 | 说明 |
|------|------|------|
| ERROR | 错误 | 程序错误 |
| WARN | 警告 | 需要注意 |
| INFO | 信息 | 一般信息 |
| DEBUG | 调试 | 调试信息 |

## 常见问题

**Q: 如何过滤错误日志？**
A: 在过滤设置中选择 ERROR 级别。

**Q: 可以自定义关键字吗？**
A: 可以添加多个关键字进行高亮。

**Q: 支持哪些日志格式？**
A: 支持常见日志格式自动识别。
