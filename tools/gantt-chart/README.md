# 甘特图生成器

> 可视化项目计划和任务时间线

## 功能特性

- **可视化** - 图形化展示项目进度
- **Markdown 导出** - 生成 Markdown 格式
- **负责人** - 分配任务负责人
- **时间范围** - 自定义开始和结束日期

## 快速开始

1. 添加项目任务
2. 设置开始日期和结束日期
3. 分配负责人
4. 生成甘特图
5. 导出为 Markdown

## 甘特图格式

```markdown
| 任务 | 负责人 | 开始日期 | 结束日期 |
|------|--------|----------|----------|
| 设计 | 张三   | 2026-03-01 | 2026-03-05 |
| 开发 | 李四   | 2026-03-06 | 2026-03-15 |
```

## 常见问题

**Q: 甘特图可以导出什么格式？**
A: 可导出 Markdown 格式，复制到支持 Markdown 的工具中使用。

**Q: 可以添加多少任务？**
A: 支持添加任意数量的任务。

**Q: 日期如何填写？**
A: 使用 YYYY-MM-DD 格式，如 2026-03-18。
