# JWT 解析器

> 解码 JWT Token，查看 Header 和 Payload 内容

## 功能特性

- **完整解析** - 显示 Header、Payload、签名
- **过期检查** - 自动检测 Token 是否过期
- **结构化展示** - JSON 格式化，易于阅读
- **无需密钥** - 前端解码，无需验证签名

## JWT 结构说明

JWT 由三部分组成，用 `.` 分隔：

```
header.payload.signature
```

- **Header** - 算法和令牌类型
- **Payload** - 包含用户信息和声明
- **Signature** - 签名（需要密钥验证）

## 快速开始

1. 粘贴 JWT Token 到输入区域
2. 工具自动解析并显示：
   - Header（算法信息）
   - Payload（数据内容）
   - 过期时间状态
3. 点击复制按钮获取特定部分

## 常见声明说明

| 声明 | 说明 |
|------|------|
| `iss` | 签发者 |
| `sub` | 主题/用户标识 |
| `exp` | 过期时间 |
| `iat` | 签发时间 |
| `name` | 用户名 |
| `email` | 用户邮箱 |

## 常见问题

**Q: 没有密钥可以解码吗？**
A: 可以解码 Header 和 Payload，但无法验证签名。

**Q: 显示「已过期」是什么意思？**
A: Token 的过期时间（exp）已过，该 Token 已失效。

**Q: Token 信息安全吗？**
A: 此工具仅在本地浏览器处理，不会上传数据。但请勿在公共场合输入敏感 Token。
